数据库的连接

发表于 2019-10-27

在学习Java的时候，我们在最后的时候都会要学JDBC数据库连接，其实书上的教程讲的很明确，看了就都能够明白了。
书上连接的是MySQL，我的电脑没有安装这个比较小的MySQL，而是安装了我上数据库课程用的Oracle 11g，所以与书上的连接还是有区别的，今天特意记录自己的连接步骤，防止以后自己再次连接错误。
连接Oracle的步骤如下：
打开了自己编写Java的集成器，我用的是IDEA，因此连接也是按照别人的教程来连接，但是踩了很多坑。
1.打开file下的project structure
2.在弹出的框中选中modules这个模块
3.然后点击添加：选中jars or directories这个选项，之后进入到自己oracle的安装路径下，找到子目录下的JDBC目录，选中其中的一个ojdbc.jar结尾的文件，点击确定。就加入数据库成功了。
4.之后用代码来连接你的数据库，才能够在Java下使用sql语句对数据库进行操作，连接代码如下：
Connection con=null;

class.forName(“oracle.jdbc.driver.OracleDriver”);//加载数据库的驱动器
String url=”jdbc:oracle:thin:@localhost:1521:orcl”; //其中的orcl是你数据库的SID那个是你之前建立数据库就已经确定的
String user=”student”; //数据库的用户名
String password=”123456”; //数据库用户的密码
con=DriverManager.getConnection(url,user,password);//获取连接
另外简洁的连接代码：
con=DriverManager.getConnection(“oracle:jdbc:thin:student/123456@localhost:1521:orcl”);

最后就可以用书上的方法进行对数据库的操作。
