—今天花了一个晚上的时间来整理自己博客的背景，虽然现在依然还是一塌糊涂。希望在后来的努力学习中自己能够更加完善的制作博客的优化。今晚主要就做了两件：一给自己的next主题添加背景图片，主要操作如下，先下载一张自己喜欢的图片，将其放到theme\next\source\images的目录下，之后在进入theme\nextsource\css_schemes\Gemini中的index.styl文件中末尾加入一下代码：body {
background:url(/images/background.jpg);
background-attachment: fixed;
}
其中background.jpg是自己刚才保存到images下的图片的名称。
之后在git下测试就能看到自己背景的效果了。
看到预览效果后觉得透明度不好，这里又要进行透明度的修改。
修改透明度的过程：打开themes\next\source\css_custom下的custom.styl文件在其中输入一下代码：.header-inner{
opacity: 0.4;
}
.main-inner {
// margin-top: 60px;
// padding: 60px 60px 60px 60px;

opacity: 0.4;

}
其中的0.4就是你要设置的透明度，这里可以根据自己的喜好来编辑透明度大小。
